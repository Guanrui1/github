    # Table 表格

    用于展示多条结构类似的数据，可对数据进行排序、筛选、对比或其他自定义操作。

    基础的表格展示用法。

    <template>
  <vol-table 
    ref="table"
    :moduleName="moduleName"
  ></vol-table>
</template>

<script>
  export default({
    data() {
      return {
        moduleName: '岗位管理'
      }
    }
  })
</script>
